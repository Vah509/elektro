Шкаф керування виготовлено для системи аерації на агропідприємстві. Об'єкт експлуатує чотири трифазних двигуни по 15 кВт, запуск яких здійснюється через плавний пуск з подальшим байпасом на пряме живлення.

Встановлено два плавних пуски CHNT RQ2 18.5 кВт — кожен обслуговує два двигуни почергово. Після розгону першого двигуна пусковий контактор відключається, байпасний підключає двигун напряму, а плавний пуск переходить до наступного. Така схема дозволяє обійтись двома пристроями замість чотирьох без втрати надійності.

На кожен двигун передбачено два контактори CHNT: пусковий (під час розгону через плавний пуск) та байпасний (для роботи напряму після розгону). Захист двигунів забезпечують автомати та триполюсні роз'єднувачі E-NEXT для безпечного обслуговування.

Логіку роботи реалізовано на ПЛК Schneider Electric Zelio з власним програмним забезпеченням. Шкаф підтримує роботу за суточним та тижневим таймером — таймер винесено окремим пристроєм для зручного програмування персоналом без доступу до ПЛК. З дверей шкафа доступне ручне керування кожним двигуном окремо через кнопки ПУСК/СТОП із сигнальними лампами стану.
